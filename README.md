# 🔐 SecureGate AI

Real-time AI-powered surveillance and tailgating detection system built using YOLOv8, ByteTrack, OpenCV, and Streamlit.

---

## 🚀 Features

- Real-time human detection
- Multi-object tracking with ByteTrack
- Entry and exit monitoring
- Tailgating detection system
- Live Streamlit dashboard
- Audio security alarms
- Event logging system
- Automatic screenshot capture
- Real-time surveillance analytics

---

## 🛠 Tech Stack

- Python
- YOLOv8
- OpenCV
- Streamlit
- Supervision
- ByteTrack
- Pygame
